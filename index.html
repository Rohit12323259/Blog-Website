World's first USB4 2.0 cables promise 80Gbps speeds

USB4, which is supported on most recently launched motherboards and devices, caps out at an impressive 40Gbps throughput. But with USB4 2.0 cables finally here, users will soon experience double those speeds and up to 240 watts (48V/5A) power delivery.

Android's latest security feature will lock out devices when detecting "motion associated with theft"

Android's latest security feature enhances device protection by automatically locking phones when it detects "motion associated with theft." This innovative capability aims to thwart unauthorized access during suspicious activities, providing users with an added layer of security. As smartphone theft continues to rise, this feature represents a proactive step by Google to safeguard user data and privacy.
